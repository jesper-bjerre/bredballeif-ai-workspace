# Bredballe IF børneattest-administrator

Skill til read-only administration af børneattester for frivillige i Bredballe IF via Conventus.

```powershell
$env:PYTHONPATH = "skills/bredballeif-boerneattest/scripts"
python -m agent --help
```

OpenClaw/Linux kan whiteliste `skills/bredballeif-boerneattest/bin/bredballeif-boerneattest.sh`.
