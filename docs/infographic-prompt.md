# Prompt til infographic om Bredballe IFs AI-agenter

```text
Lav en simpel, professionel infographic på dansk om Bredballe IFs Telegram-bots og OpenClaw AI-agenter.

Formål:
Vis bestyrelsen og Padel-udvalget, hvordan de kan bruge Telegram-bots til at automatisere administrative arbejdsopgaver.

Format:
- Én side / ét billede
- Rent og overskueligt layout
- Dansk tekst
- Brug Bredballe IF som kontekst
- Stil: foreningsvenlig, moderne, enkel, ikke for teknisk
- Gerne to tydelige kolonner eller to sektioner

Titel:
"AI-agenter i Bredballe IF"

Undertitel:
"Bestyrelse og udvalg kan bruge Telegram-bots til at automatisere administrative opgaver"

Sektion 1:
Telegram bot: "BIF Administrator"
OpenClaw-agent: "bredballeif-administrator"
Adgang: Kun bestyrelsen i Bredballe IF

Vis muligheder:
- Hjælper med at indhente børneattester for nye frivillige
- Hjælper bestyrelsen med kontrol af børneattester for ca. 300 frivillige
- Kan lave overblik, status og opfølgningslister
- Reducerer manuelt arbejde og risiko for glemte opgaver

Sektion 2:
Telegram bot: "BIF Padel Administrator"
OpenClaw-agent: "bredballeif-padel-administrator"
Adgang: Kun Padel-udvalget i Bredballe IF

Vis muligheder:
- Håndterer medlemmer i Conventus og HalBooking
- Hjælper med medlemsopslag og medlemsstatus
- Ved arrangementer kan agenten hjælpe med:
  - booking af baner
  - oprettelse af tilmeldingsmulighed i Conventus
  - udkast til SoMe-opslag
- Gør gentagne padel-opgaver hurtigere og mere ensartede

Vis gerne flowet:
"Bestyrelse / udvalg" → "Telegram bot" → "AI-agent" → "BIF-systemer og opgaver"

Vigtige noter nederst:
- Bots er kun for godkendte brugere
- Secrets og persondata håndteres sikkert
- Agenten bruger godkendte skills og whitelisted funktionalitet

Brug ikoner:
- Telegram/chat
- bestyrelse/gruppe
- tjekliste
- lås/sikkerhed
- kalender/arrangement
- medlemsdatabase

Farver:
Brug en rolig forenings-/idrætsforeningsstil med blå, grøn og hvid. Undgå tung teknisk DevOps-stil.
```
