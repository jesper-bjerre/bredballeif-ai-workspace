# Prompt til overordnet infographic om OpenClaw og Telegram

```text
Lav en enkel infographic på dansk med titlen:

"Sådan hænger AI-agenterne sammen"

Formål:
Vis på en let forståelig måde hvordan Bredballe IFs bestyrelse og udvalg bruger Telegram til at tale med AI-agenter i OpenClaw.

Stil:
- Meget lidt tekst
- Overskueligt flow
- Moderne, rolig og foreningsvenlig visuel stil
- Ikke teknisk tung
- Én side / ét billede
- Brug ikoner frem for lange forklaringer

Vis dette flow:

1. Brugere
   - Bestyrelse
   - Padel-udvalg

2. Telegram
   - BIF Administrator
   - BIF Padel Administrator

3. OpenClaw
   - Kører AI-agenterne
   - Styrer adgang og godkendte værktøjer

4. AI-agenter
   - bredballeif-administrator
   - bredballeif-padel-administrator

5. Skills og systemer
   - Børneattester
   - Conventus
   - HalBooking
   - Padelbaner
   - Arrangementer

Kort tekst på billedet:
- "Brugerne skriver i Telegram"
- "OpenClaw sender beskeden til den rigtige AI-agent"
- "Agenten bruger godkendte skills"
- "Resultatet sendes tilbage i Telegram"

Vis adgang:
- Bestyrelse → BIF Administrator
- Padel-udvalg → BIF Padel Administrator

Vis gerne med pile:
Brugere → Telegram → OpenClaw → AI-agent → Skills/systemer → Telegram-svar

Ikoner:
- Personer/gruppe
- Telegram/chat
- Server/cloud for OpenClaw
- Robot/AI-agent
- Tjekliste
- Kalender
- Database
- Lås/sikkerhed

Farver:
Brug blå, grøn og hvid. Hold det lyst, enkelt og tillidsvækkende.

Undgå:
- Lange tekstbokse
- Kode
- Tekniske filstier
- Detaljerede kommandoer
```
